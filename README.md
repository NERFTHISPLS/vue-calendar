# Vue 3 Calendar Component
Реактивный календарь на Vue 3 с TypeScript, поддержкой:
- Выбора даты с подсветкой выбранного дня
- Переключения месяцев
- Локализации названий месяцев и дней недели (смена языка)
- Задания начальной даты

## Установка

1. Скачать все файлы из репозитория
2. 
```
npm install
```
3.
```sh
npm run dev
```
